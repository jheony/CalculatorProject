## 주요 기능

---

- Lv 1. 기본 사칙연산 및 반복
    - 숫자 2개와 사칙연산 문자 1개를 입력받아 연산
    - 오류 예외 처리
    - 'exit' 입력 전까지 계산 무한 반복
